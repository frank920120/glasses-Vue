# Frank 眼镜商城

## 简介

```
这是一个用vuejs所完成的前端开发项目
```

## 安装依赖

```
npm install
```

## 项目启动

```
npm run serve
```

## 技术

```
1.vuejs
2.vue-router
3.vuex
4.vant
```
